# 🕵️ దొంగ ని కనిపెట్టు (Donga Ni Kanipettu)

A fun multiplayer imposter word game — played offline with friends!

Everyone gets the same word to describe, except the imposter. Find out who the imposter is!

## How to Play
1. Create a game & add player names
2. Share the link with friends
3. Everyone taps their name to enter
4. Host clicks **New Word** → everyone clicks **Refresh**
5. Describe the word, discuss & find the imposter 🕵️

## Run Locally
```bash
npm install
node server.js
# Open http://localhost:3000
```
